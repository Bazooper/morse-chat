# Morse Chat

## Prerequisites
### Server
- Rust
- Cargo
- C Linker

### Client
- Rust
- Cargo
- [Tauri dependencies for the platform](https://tauri.app/start/prerequisites/)

## How to build
Execute following commands:
```sh
cargo build
```

## How to run
### Server
Execute following command
```sh
cargo run -p morse-server
```

### Client
Execute following command
```sh
cargo run -p morse-client
```
